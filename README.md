# wanglai 往来

## 每一份人情往来都值得珍惜
这是一个 Android 项目，读入 Excel 表格来快速查看人情往来。应用市场有很多类似的产品，但是很多因为要盈利需求就会有广告，但我想要一个纯净版，即便只是一个功能简易的工具。不依赖 WPS 这些原因是手机屏幕太小了，手机直接查看 Excel 表格观感和操作不太好。

## Excel 文件格式

App 只读取**第一个工作表**的前四列，列顺序固定：

| 列 | 字段 | 说明 |
| --- | --- | --- |
| A | 姓名 | 必填；为空的行会被跳过 |
| B | 地址 | 任意文本，参与搜索 |
| C | 金额 | 任意文本；整数会自动去掉 Excel 默认追加的 `.0` |
| D | 页码 | 自动提取其中的数字，"第 3 页" / "P5" / "3" 都会显示为 `3` |

示例：

| 姓名 | 地址 | 金额 | 页码 |
| --- | --- | --- | --- |
| 张三 | 北京市朝阳区 | 500 | 第 1 页 |
| 李四 | 上海市浦东 | 1000 | P2 |
| 曾广莲 | 成都市青羊区 | 800 | 3 |

说明：

- 文件需是 `.xlsx`（OOXML，ZIP 包），不支持老式 `.xls`。
- 首行可有可无：如果第一行 A 列是 `姓名`、`名字`、`name` / `Name` / `NAME` 之一，会被识别为表头并跳过；否则按数据处理。
- 多出的列会被忽略，少于四列的行也能读，缺失字段显示为 `—`。
